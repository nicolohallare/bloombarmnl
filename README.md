# Bloom Bar MNL — Site

Self-contained static site for bloombarmnl.com.

## Files
- `index.html` — the full page
- `robots.txt` — crawler instructions
- `sitemap.xml` — single-page sitemap
- `images/` — all photos (self-hosted, no SiteGround dependency)

## Updating
- Edit any file directly via GitHub web (pencil icon → commit)
- Netlify auto-deploys within 30 seconds
- To change a photo, drop a new file in `images/` keeping the same filename
